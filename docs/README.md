# KCP Tube

## 模式介绍

### 支持的模式
目前支持 3 种模式：
- 客户端模式
- 服务端模式
- 中继节点模式

## 用法

### 命令行
`kcptube config.conf`
或
`kcptube config1.conf config2.conf`
如果有多个文件，可以接着补充。
