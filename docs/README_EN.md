# KCP Tube

## Modes

### Supported Modes
Currently 3 modes are supported:
- Client Mode
- Servers Mode
- Relay Mode

## Usage

### Command
`kcptube config.conf`
or
`kcptube config1.conf config2.conf`
If there are multiple files, you can then add them.
